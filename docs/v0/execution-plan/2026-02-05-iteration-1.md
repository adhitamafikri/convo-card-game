### Phase 2

> This phase focuses on page and components creation

- [x] Create these reusable components in `components/` directory
  - [x] Header
    - Specs:
      - Full width, height of 68px, padding of 1em, flex row, items centered vertically
      - Contains the title of the game on the left side
  - [x] Footer
    - Specs:
      - Full width, height of 68px, padding of 1em, flex row, items centered vertically
      - Contains a simple copywriting that says "created by Adhitama Fikri @kjonexa", centered horizontally and vertically
  - [x] Input, based on the daisyUI `Input` component
  - [x] Button, based on the daisyUI `Button` component
  - [x] Modal, based on the daisyUI `Modal` component
  - [x] GameCard
    - Based on the daisyUI `Card` component
    - Expected Behaviors:
      - This component will be used to display the content of the card that is being drawn or selected
      - isOpen is false when the card is just drawn from the deck, the content will be hidden
      - isOpen is true when the card is picked by the player, the content will be displayed
      - onSelect will be the click handler on the component itself, allowing player to reveal the content of the selected card
  - [x] CardStack
    - Based on the daisyUI `List` component
    - Expected Behaviors:
      - The component would display the remaining cards in the stack
      - The component can draw cards to the `GameDeck`
  - [x] GameDeck
    - Based on the daisyUI `List` component
    - Expected Behaviors:
      - Displays `GameCard` drawn from the `CardStack`
      - Allows the player to pick one card and display the selected card content for them
